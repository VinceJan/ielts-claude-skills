# 雅思评分标准详表

## 写作评分标准（四维）

### Task 2 — Task Response（TR）25%

| Band | 标准 |
|------|------|
| 9 | 完全回应所有要求，立场清晰，论点充分展开且有深度支持 |
| 8 | 回应所有要求，立场清晰，论点充分展开，偶有过度概括 |
| 7 | 回应了所有部分，立场清晰，论点充分展开，但偶尔过度概括 |
| 6 | 回答了题目但部分论点不够充分，结论可能不清晰 |
| 5 | 只部分回答了题目，论点有限，可能跑题 |
| 4 | 未能回应所有部分，立场不明，论点有限且缺乏展开 |

**关键检查点：**
- 是否回答了题目的**每个**部分（漏答直接降到 5）
- 立场是否从头到尾一致
- 论点是否有具体展开（不是只说一句概括）
- 是否有具体例子支持论点

### Task 2 — Coherence & Cohesion（CC）25%

| Band | 标准 |
|------|------|
| 9 | 信息传递毫不费力，衔接自然到几乎不被注意 |
| 8 | 逻辑清晰，衔接自然，段落组织合理，偶有过度使用连接词 |
| 7 | 逻辑清晰，衔接自然，段落组织合理，偶尔过度使用连接词 |
| 6 | 有逻辑但衔接有时机械，段落内可能缺少连贯性 |
| 5 | 逻辑不够清晰，段落组织混乱，连接词使用不当 |

**关键检查点：**
- 段落之间是否有逻辑递进（不是并列堆砌）
- 连接词是否自然（However/Moreover/Furthermore 过度使用 = 机械感）
- 每段是否只说一件事
- 指代是否清晰（this/it/they 指代明确）

### Task 2 — Lexical Resource（LR）25%

| Band | 标准 |
|------|------|
| 9 | 灵活精准使用词汇，自然且 sophisticated |
| 8 | 词汇量大，能灵活使用不常见词汇，偶有搭配错误 |
| 7 | 词汇量足够，能灵活使用不常见词汇，偶尔有搭配错误 |
| 6 | 词汇基本够用，尝试使用不常见词汇但有时不准确 |
| 5 | 词汇有限，经常重复，搭配错误较多 |

**关键检查点：**
- 同一个词是否重复超过 3 次
- 是否有同义替换
- 搭配是否正确（make a decision ✓ / do a decision ✗）
- 拼写错误数量

### Task 2 — Grammatical Range & Accuracy（GRA）25%

| Band | 标准 |
|------|------|
| 9 | 多种句型灵活使用，语法和标点几乎无误 |
| 8 | 多种复杂句型，错误极少且不影响理解 |
| 7 | 使用多种复杂句型，错误少且不影响理解 |
| 6 | 混合使用简单句和复杂句，有语法错误但不频繁 |
| 5 | 句型有限，错误频繁，部分影响理解 |

**关键检查点：**
- 是否全是简单句 → 需要加入定语从句、条件句、被动句
- 主谓一致
- 时态一致
- 冠词错误

### Task 1 — Task Achievement（TA）25%

| Band | 标准 |
|------|------|
| 9 | 完全覆盖所有要求，突出关键特征 |
| 8 | 覆盖所有要求，有效突出关键特征 |
| 7 | 覆盖关键趋势和数据，有概述 |
| 6 | 覆盖了要求但可能遗漏关键数据 |
| 5 | 只部分覆盖，可能有误解 |

### 评分计算

- Task 1 和 Task 2 各自独立评分
- **Task 2 权重是 Task 1 的 2 倍**
- 总分 = (Task 1 分数 × 1 + Task 2 分数 × 2) ÷ 3
- 四舍五入到最近的 0.5

## 总分计算

总分 = 四科平均值，就近取整到最近的 0.5 或整数。

**取整规则（官方规则：.25 向上到 .5，.75 向上到下一个整数）：**
- x.00 - x.24 → x.0（如 6.125 → 6.0）
- x.25 - x.74 → x.5（如 6.375 → 6.5，6.625 → 6.5）
- x.75 - x.99 → x+1.0（如 6.875 → 7.0）

**策略含义：**
- 目标 7.5 = 听力 8 + 阅读 8 + 写作 6.5 + 口语 6.5（29 ÷ 4 = 7.25 → 7.5）
- 目标 7.0 = 听力 7.5 + 阅读 7.5 + 写作 6 + 口语 6（27 ÷ 4 = 6.75 → 7.0）

**AI 评分校准：** AI 评分只能作为训练反馈，不是官方成绩。输出时优先给证据和区间；重要节点建议用官方样题、考官评语、模考或多个独立工具交叉校准。
