## 雅思阅读备考路径（四步法）

### 传统方式的问题

**为什么之前提分慢：**
1. **背单词**：忘记词需要手动记录
2. **长难句**：看不懂，没有老师讲，就算有老师也只是讲一次
3. **错因分析**：遇到长难句不懂错的原因
4. **精读效率低**：一篇精读3小时，但其实还是没有理解
5. **第二天从零开始**：第二天还是忘记

### AI时代的优势

**有了Claude Code和雅思Skill后：**
- 步骤自动化
- 相当于陪跑老师
- 提分速度比之前快一半

**具体帮助：**
- 帮记单词
- 帮分析长难句
- 帮分析错因
- 帮把这些东西全部记录下来

## 六分到6.5分卡住的原因

**核心问题：**
1. 词汇记不住
2. 长难句不懂

**根本原因：**
- 一直在刷题，但没有去精读题
- 做完了不沉淀
- 下一篇从零开始
- 一直刷一直错，一直刷一直错

**解决思路：**
- 不要做很多题
- 只需要做好每一套题
- 每一套阅读都走四步
- 让Claude Code帮你做

## 四步法详细拆解

### 第一步：背单词

**推荐方法：** 场景 + 长难句反考

**推荐教材：** 雅思词汇真经

**为什么用这本书：**
- 词汇按场景分类（经济学、哲学、历史等）
- 雅思考试也是按场景考
- 每篇文章从头到尾讲一个主题
- 主题词汇分布和背的场景词汇重合
- 背起来熟悉，越做越有信心
- 越有信心越做越快，正确率越高

**具体操作：**
1. 让Claude Code出15个词汇（如自然地理）
2. 先背这些词汇
3. 让AI生成5个雅思风格的长难句（4个词一组）
4. 有场景就记住词了，同时练了长难句
5. 看不懂的长难句让AI解释
6. 长难句里有其他核心词，顺便学习

**效果：**
- 练完一套可以一天背50个左右的词
- 基于场景背，不会忘记
- 这是正确背单词的姿势

### 第二步：做题技巧

**题型分类：**
1. **填空题**：考词汇量，看能否定位词并换掉
2. **判断题**：考长难句理解
3. **选择题**：考长难句理解
4. **匹配题**：考定位词和长难句理解

**判断题难点：**
- "错"和"没有说"的区分
- 考能否精准找到题所在的句子
- 理解句子意思

**选择题做法：**
1. 先看题干
2. 回去找定位
3. 看相关句子
4. 回来做题

**核心：** 都是对长难句的考试

**不要觉得难：** 提升很快，靠第三步和第四步

### 第三步：对答案与错题分析

**操作流程：**
1. 把题本身拍照
2. 文章拍照
3. 答题记录拍照
4. 正确答案拍照
5. 四个东西一股脑喂给Claude Code

**AI分析内容：**
- 定位词
- 同义替换
- 干扰项
- 重新陷阱
- 每个错题具体错在哪里
- 可能的原因
- 定位词所在句子的意思

### 第四步：精读文章

**操作流程：**
1. 第一次读文章读不懂
2. 把所有生词圈出来
3. 长难句画杠画出来
4. 拍照片发给Claude Code

**AI自动记录：**
- 记这些长难句什么意思
- 记你不认识的词到知识库
- 把生词写到本地Markdown文件（错词本/生词本）
- 把长难句放到另一个文件（长难句错题本）

**长难句文件结构：**
- 长难句本身
- 分析
- 结构拆解
- 几个类似句子
- 整体是一个"原子"

**核心价值：**
- 自动建立自己的错题本
- 之前建立成本很高，要自己写
- 现在自动帮我们把这些生词写到本地文件
- 相当于有了自己的做题历史和知识库
- 错题库从脑子里外化显化到本地文件
- Claude Code每次都可以读，省精力
