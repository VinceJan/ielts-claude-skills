# 用CC/Codex自学雅思阅读八周冲8分全打法

> 来源: 抖音 | 作者: YAN | 视频ID: 7647849072598060297

## 视频概述

**核心目标：**
- 八周时间从6分冲到8分或8.5分
- 包括如何用Agent（Claude Code或Codex）的用法

**核心价值：**
- 完全可以代替市面上的任何雅思大班、小班、一对一班
- 用AI学习，八周把阅读从6提升到8

## 雅思阅读考试结构

### 基本信息
- 总时间：60分钟左右
- 三篇文章，一篇文章20分钟
- 最后一篇比较难
- 一共有40道题
- 每篇大概800-1000词

### 时间分配建议
- 第一篇文章：15分钟做完
- 第二篇文章：15-20分钟
- 最后一篇文章：留20多分钟

### 话题分布
- 科技
- 教育
- 历史
- 生物
- 人文社科

### 8分以上容错率
- 最多错5道题
- 第一篇错1道
- 第二篇最多错2道
- 第三篇最多错2道
- 第一篇不错，后面容错率多一些

## 第一步：词汇

### 核心认知
- **做阅读主要考两个地方：词汇和长难句**
- 大部分细节题、同义替换题、填空题都在考对定位词或重点词的理解
- **词汇非常重要**

### 推荐教材
- 雅思词汇真经
- GitHub上有人总结成了库（PDF + 官方音频）

### Claude Code和Codex的作用
- 已经有了PDF
- 帮你排每天应该复习哪些词
- 每个词配一个语境
- **配语境背词比干背快很多**
- 第二天主动测试昨天背的词有没有不理解或忘记的

### 生词库扩展
- 做阅读时遇到新词丢进去
- 作为不断增长的知识库
- 词库不光是雅思词汇，还包括做真题时遇到的生词

## 第二步：真题练习

### 真题选择
- 雅思现在出到了20（从1到20）
- 1到10：比较简单、比较古老
- **建议从15到20练**

### 真题数量
- 15到20每个剑桥雅思真题包括4个Test
- 4个Test = 4套阅读
- 一共24个Test = 24套阅读

### 练习安排
- 每周练2-3套
- 分两个部分：
  - **15到18**：备考时做（16套阅读）
  - **19到20**：考前最后两周做（8套阅读）

### 核心原则
- **不需要做特别多的阅读题**
- **最重要的是精读，而不是多做**

## 第三步：精读方法（四步法）

### 第一步：计时做完整套
- 做不完没关系，但一定要计时
- 知道现在的水平在哪里
- 做的过程中肯定会有卡顿

### 第二步：不看答案，标生词
- 第二遍看的时候不要对答案
- 把生词全部画出来
- 拍照丢给Agent（Claude Code）
- 用雅思Skill放进库里
- 下次调用雅思部分时，把词也入生词库去检测
- 下次背单词时，雅思Skill就知道这些词在生词库里，会帮你纳入测验

### 第三步：标长难句
- 把长难句画出来
- 复制或截图丢给Agent
- 让AI帮你拆解长难句

### 第四步：对答案
- **把生词和长难句画完之后，再去对答案**

## 长难句分析

### 遇到的问题
- 句子太长，完全不知道里面说了什么
- 不理解谁做了什么、在什么情况下做的

### 解决方法
- 一句一句去啃
- 有AI帮助会相对轻松

### AI具体帮助
- 分析句子结构
- 写出五个结构一样的类似句子
- 在对比、类比、解释的情况下定制化学习
- 很快去理解

### 学习建议
- **不要图快**
- 今天可以只学这一种长难句或这一句长难句
- 不要把这一句看的差不多了再去做下一个
- 大部分长难句结构基本都在那几个分类里
- **把这一句啃透了，这一类的长难句都可以啃透**
- 不要囫囵吞枣的读完这个再去读下一个

### Claude Code和Reading Skill的作用
- 帮你解释
- 帮你出类似句型
- 把句型变成考题
- 把长难句复制给Reading Skill
- 让它把结构拆开
- 给你出中英对照译文
- 给你几个同结构句链

### 学习效果
- 看了大概5个同样结构的句型后
- 下次再看这个长难句，复习这篇阅读时
- 可以很快理解句子意思
- 不需要记很复杂很古板的句子结构
- 完全可以靠语感
- 读完这篇长难句，用AI练几个同类型长句
- 下次看到直接就可以看懂

### 核心理念
- **目的是把句子看懂，而不是把每一部分都拆解**
- 语法对不对不重要
- 只需要把句子看懂就可以了

## 第四步：四类题型答题技巧

### 1. 填空题（最简单）

**特点：**
- 读原文时就开始做这道题
- 题目顺序和原文基本一致
- 答案可以直接从原文抄
- 词也可以用一样的

**效果：** 读完原文时，基本上这道题已经做完了，节省时间

### 2. 判断题

**核心难点：** False和Not Given的区分
- **False**：说了，但是错的
- **Not Given**：没有提这件事

**蒙题技巧（实在没招时用）：**
- 看到only、always、never等敏感词
- 如果没有时间看原文或真的看不懂
- 基本上是一个False

**建议：** 还是要真的读懂原文，这种蒙的情况是实在没招了再用

### 3. 选择题

**做题流程：**
1. 先理解题干
2. 回头看原文
3. 做完填空题和判断题后，再来到选择题
4. 看选择题题干
5. 去原文里找题干可能在哪一段
6. 找完那段后回来看选项

**核心技巧：**
- 选项、题干和原文有很多词或句子是重复的或相似的
- 看完题干回去看原文，再看选项时会很熟悉
- 可以直接定位到那个句子

**常见错误：**
- 很多同学错在"我觉得这个听起来好像是对的"
- 往往这种就是迷惑项
- **要找到原文明确说的，而不是听起来对的**

**核心考察：** 定位词和理解原文的能力，阅读反反复复都在考这个

### 4. 匹配题（最难）

**尤其是标题匹配：**
- 要求读的时候就理解每一段内容
- 画出来主旨句

**技巧：**
1. 先挑简单的段落开始做
2. 主旨句一般出现在段落的第一句或最后一句
3. 看连接词后面那句（如however、even though后面）
4. 主旨句后面通常有一句解释主旨句的案例或论点

**总结：** 先做简单的段，难的段先看首句和尾句，或者看连接词后面那句

## 错题分析与可视化

### AI帮助分析
- 分析自己到底哪一类题做的比较多
- 分析自己错的哪种陷阱模式容易做错

**常见错误模式举例：**
- 原文说了A，题目问A+B时容易错
- 说明习惯性自己臆想一些原文不存在的内容
- 原文用比较级，题目用最高级
- 没有成功定位导致错题

### 可视化数据
- Claude Code和雅思Skill可以帮助统计
- 包括可视化数据板
- 针对性练习
- 比盲刷有用得多

## 总结

**雅思阅读四部分：**
1. **词汇**：通过AI帮助学习
2. **真题**：只需要24套到30套Test就足够了
3. **精读**：每一套一定要进行精读
4. **做题技巧**：四类题型对应技巧

**核心承诺：** 这四件事情按照我说的去做，用AI自学雅思八周时间提升2-2.5分完全没有问题
