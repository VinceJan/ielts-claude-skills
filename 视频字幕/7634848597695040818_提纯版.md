# 用CC雅思Skill 3.0不报班自学雅思7.5具体操作流程

> 来源: 抖音 | 作者: YAN | 视频ID: 7634848597695040818

## 版本更新与使用情况

**背景：**
- 20天前分享如何用Claude Code 4小时考到7.5分
- 已有100多个同学使用雅思Skill考雅思

**核心目标：**
- 任何人用这套雅思Skill自学雅思
- 不需要报班
- 一个月可以提升1-1.5分

## 诊断与规划

### 水平模拟
- 做一套真题
- 用IELTS Coach和IELTS Diagnose两个Skill
- 把做过的真题直接传给这两个Skill

**输入方式：**
- 图片或文字信息都可以
- 直接截图也没问题
- 让AI帮你分析、诊断、规划

### 诊断功能
- 了解你的情况和进度
- 给出建议
- 分析成绩，定位弱项
- 把数据记录在文件夹中
- 制定针对性计划（相当于雅思班的班主任）

## 口语训练方法

### 正确使用方法
1. 找最近的10套雅思真题
2. 提取口语问题（Part 1、Part 3）
3. 把这些问题喂给Claude Code

### 生成个人故事
- 想一下这些问题用中文怎么回答
- 整合自己的过去经历
- 把中文回答和过去经历一起喂给AI
- 生成5个万能故事

**核心优势：**
- 故事同时覆盖考点和自己的真实经历
- 记自己的故事比网上找的故事容易得多
- 本身就是自己的经历，只是翻译成英文

## 写作训练方法

### 四维评分批改
- 左边放原文，右边放Claude Code界面
- 写完原文后喂给Claude Code
- 进行四维评分的批改

**四维评分（雅思官方评分标准）：**
1. **Task Response** - 是否扣题
2. **Coherence & Cohesion** - 逻辑通顺
3. **Lexical Resource** - 词汇量
4. **Grammatical Range & Accuracy** - 语法

### 改写对比功能
- 改范文时，原文附在括号里
- 清晰看到范文和原文的区别
- 理解为什么范文写得更好
- 识别口语化表达 vs 书面化表达

### 学习记录可视化
- 做完写作后让Claude Code记住答题记录
- 后续可以可视化整个复习过程
- 不需要自己记，可以可视化2个月的课题分布和进步

## 阅读与词汇结合训练

### 核心理念
- **阅读和词汇可以一起学**
- 每天背词汇本身非常痛苦且遗忘率高
- **最好的背词汇方法：结合阅读去背**

### 操作流程
1. 做阅读遇到不懂的词
2. 这些词是雅思核心词，也是你不懂的词
3. 提取这些词放到词汇库
4. 第二天自动推给你背诵

### 背词技巧
- 不要一个一个单独背
- 让Skill把100个词串成一个故事
- 比如组成生态链的故事或上学的故事
- 记住故事后就知道每个词在语境中的意思

**额外收获：**
- 背完词后，阅读长难句能力也提升了
- 词语境中设定了长难句
- 不懂长难句可以立刻问Claude Code

## 听力训练方法

### 错题分析
- 听完后不懂的内容，AI帮助分析
- 分析错在什么地方
- 如果10篇听力都错在某个词或表达，着重复习那个部分

### 精听技巧（重要）

**核心原则：**
- **不需要听很多文章**
- **要把一篇文章听得很熟练**

**训练方法：**
1. 放1.25倍速或1.5倍速
2. 第一次放1.5倍速可能完全听不懂
3. 一直练到放1.5倍速能听懂每段讲什么
4. 理解行为逻辑、生僻词位置

**目标：** 听力达到7.5分或8分的水平

## 3.0版本核心：本地数据闭环

### 系统架构
- 每次和Claude Code做完交互
- Claude Code把答题记录、错题记录在本地索引
- 用Dashboard可视化

### 核心价值
- **相当于一个完整的本地备考引擎**
- 所有雅思一对一班或大班要花几万块钱做的事情
- 这个Skill和Claude Code完全可以帮你复现
- **不需要任何其他外部教练**
