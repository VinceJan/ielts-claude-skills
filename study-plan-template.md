# 8 周雅思备考计划模板

基于作者 YAN 的实战经验（两个月从 6 到 7.5），结合四步法精读和输入输出两阶段写作训练。

---

## 总体策略

- **默认优先听力和阅读**（总分拉分效率高，反馈更客观）
- **有单项小分要求时优先补最低单项**，不要机械套用固定比例
- **每天学习 2-3 小时**
- **听力阅读隔天交替，写作口语每天都有**

---

## 第 1 周：诊断 + 摸底

| 日 | 科目 | 任务 | 时长 |
|----|------|------|------|
| 一 | 摸底 | 做一套完整真题（听读写 + 口语录音） | 3h |
| 二 | 诊断 | 用 `/ielts-diagnose` 分析成绩 | 30min |
| 三 | 词汇 | 背第 1 组 15 词 + 语境长难句 | 45min |
| 四 | 阅读 | 做第 1 套 + 四步法精读 | 1h45min |
| 五 | 听力 | 做第 1 套 + 精听错题 | 1h30min |
| 六 | 写作 | 背 1 篇范文 + 拆结构 | 1h |
| 日 | 口语 | 用 `/ielts-speaking` 生成万能故事 | 1h |

**本周目标：** 知道自己几分、差在哪、该往哪使劲

---

## 第 2-3 周：基础夯实期

### 每日固定

| 时间段 | 任务 | 时长 |
|--------|------|------|
| 上午 | 词汇：15 新词 + 复习旧词 | 45min |
| 下午 | 主科目训练（见下表） | 1h30min |
| 晚上 | 口语：10 道 Part 1 或 Part 2 | 30min |

### 每周安排

| 日 | 主科目 | 具体任务 |
|----|--------|---------|
| 一 | 听力 | 做一套 + 精听错题 + 1.25x 跟读 |
| 二 | 阅读 | 做一套 + 四步法精读（标生词+长难句） |
| 三 | 听力 | 做一套 + 精听错题 + 1.25x 跟读 |
| 四 | 阅读 | 做一套 + 四步法精读（标生词+长难句） |
| 五 | 听力 | 做一套 + 精听错题 + 1.25x 跟读 |
| 六 | 写作 | 背范文 + 拆结构 + 举一反三（输入期） |
| 日 | 写作+口语 | 限时写一篇 + `/ielts-writing` 批改 |

**本周目标：** 词汇量 +200，阅读正确率提升 10%，听力精听覆盖 5 篇

---

## 第 4-5 周：强化提升期

### 变化点
- 听力在正常速度稳定理解后，再用 1.25x → 1.5x 做压力训练
- 写作从输入期 → 输出期（开始自己写）
- 阅读开始练题型技巧（判断题/匹配题专项）
- 口语开始练 Part 3 框架

### 每周安排

| 日 | 主科目 | 具体任务 |
|----|--------|---------|
| 一 | 听力 | 做一套 + 答案句精听 + 错题模式分析 |
| 二 | 阅读 | 做一套 + 四步法 + 题型专项（判断题） |
| 三 | 听力 | 做一套 + 答案句精听 + 错题模式分析 |
| 四 | 阅读 | 做一套 + 四步法 + 题型专项（匹配题） |
| 五 | 听力 | 做一套 + 答案句精听 + 错题模式分析 |
| 六 | 写作 | 限时写 Task 2（20min）+ 批改 + 背改写版 |
| 日 | 写作+口语 | 限时写 Task 1（15min）+ Part 3 框架练习 |

**本周目标：** 写作从输入转输出，阅读各题型都有策略

---

## 第 6-7 周：冲刺期

### 变化点
- 开始做 Cambridge 19-20（最新真题，留到考前）
- 写作每天一篇，批改后背改写版
- 听力正常速度稳定做对；1.25x/1.5x 只做压力训练
- 口语用 ChatGPT Voice / Gemini Live 模拟考

### 每周安排

| 日 | 主科目 | 具体任务 |
|----|--------|---------|
| 一 | 听力 | C19 套题 + 答案句精听 + 错题回顾 |
| 二 | 阅读 | C19 套题 + 四步法 + 高频错题型复习 |
| 三 | 听力 | C19 套题 + 答案句精听 + 同义替换复习 |
| 四 | 阅读 | C19 套题 + 四步法 + 长难句复习 |
| 五 | 听力 | C19 套题 + 答案句精听 + 生词测试 |
| 六 | 写作 | 限时写 + 批改 + 重点复习高频错误 |
| 日 | 全科 | 模拟考试（严格计时）+ 复盘 |

**本周目标：** 接近考试节奏，查漏补缺

## 每日时间分配参考

| 任务 | 时长 | 说明 |
|------|------|------|
| 词汇 | 30-45min | 20min 新词 + 20min 复习 + 5min 入库 |
| 主科目 | 60-90min | 听力/阅读交替 |
| 口语 | 20-30min | 每天练一点，不集中突击 |
| 写作 | 60min | 输入期背范文 / 输出期限时写 |
| **总计** | **2.5-3.5h** | |

---

## 关键原则

1. **精读 > 刷量** — 一套题做透比十套题刷过有用
2. **语境背词 > 孤立背词** — 结合阅读背词记得牢
3. **定位错因的精听 > 大量泛听** — 先处理答案句、弱读、干扰项和同义替换
4. **背改写版 > 自己瞎写** — 输入够了才能输出好
5. **数据驱动 > 感觉驱动** — 用 `/ielts-dashboard` 看数据，不靠感觉
