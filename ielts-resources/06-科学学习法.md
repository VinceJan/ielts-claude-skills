# 雅思备考科学学习方法

> 基于认知科学研究和雅思备考实践。结论按“稳健原则 + 可执行默认值”使用。

---

## 1. 间隔重复（Spaced Repetition）

### 科学依据

- **Nakata & Elgort 2021**（SAGE Journals, Studies in Second Language Acquisition）：间隔学习在 meaning recall 和 form-meaning matching 测试上优于集中学习
- **Nakata 2015**（Cambridge Core）：扩展间隔可能有优势，但该方向研究并不支持把它写成绝对规律
- **iJIM 研究**：60 名 ESP 学生，两种间隔都提升词汇，但仅扩展间隔显著提升学业表现

### 最佳实践

**扩展间隔规则：**
- 新词：当天背 → 第 2 天测试 → 第 4 天 → 第 7 天 → 第 14 天
- 掌握标准：连续 3 次测试正确 → 标记为已掌握
- 工具：Anki / Quizlet / Skill 内置间隔重复

**重要限制：**
- 间隔重复对显性知识（meaning recall）有效
- 对隐性知识（semantic priming）效果不显著
- 需要配合语境使用才能真正内化

## 2. 语境背词（Contextual Vocabulary Learning）

### 科学依据

- 人类在语境中理解词义比孤立背词快 3-5 倍
- 语境提供词义的多个线索（搭配、语法位置、语义场）
- 孤立背词容易遗忘，因为缺乏记忆锚点

### 最佳实践

**正确做法：**
- 每个词配一个雅思风格的例句
- 从阅读/听力中提取生词（有真实语境）
- 把 10-15 个词串成一个故事
- 场景分类（经济学/自然科学/社会科学）

**错误做法：**
- 背单词表（abandon, ability, abnormal...）
- 一次背太多（每天 15-20 个新词最佳）
- 只看中文意思，不看英文例句

## 3. 精读 vs 刷量（Intensive vs Extensive Reading）

### 科学依据

- 精读（逐句理解 + 分析结构）比泛读更能提升阅读理解能力
- 刷量但不沉淀 = 每次从零开始，没有积累
- 长难句理解能力是阅读 6→8 的关键瓶颈

### 最佳实践

**四步法精读（推荐）：**
1. 计时做题 → 知道真实水平
2. 标生词（不对答案）→ 积累词汇
3. 标长难句 → 拆解结构 + 同结构练习
4. 对答案 → 错题分析 + 同义替换提取

**关键原则：**
- 一套题做透 > 十套题刷过
- 每套题花 1h45min（做题 60min + 精读 45min）
- 长难句要啃透，不要囫囵吞枣

## 4. 倍速精听（Speed-Adjusted Listening）

### 科学依据

- 变速听力训练的直接实证支持有限
- 但间接证据：适度提速可作为压力训练；不能保证直接对应某个 IELTS band
- 类似体育训练中的超负荷原理

### 最佳实践

**1.25x → 1.5x 递进：**
1. 正常速度做题 → 记录错题
2. 精听错题部分（反复 3-5 遍）
3. 看 Audio Script 跟读答案句
4. 1.25x 跟 Audio Script 听
5. 1.5x 跟 Audio Script 听
6. 1.5x 裸听

**关键原则：**
- 不需要听很多，把一篇听到滚瓜烂熟
- 目标：正常速度稳定理解并做对；1.25x/1.5x 作为进阶压力训练
- 听力和阅读隔天交替

## 5. 写作输入输出两阶段

### 科学依据

- 语言习得的输入假说（Krashen）：可理解输入是语言习得的必要条件
- 输出假说（Swain）：输出迫使学习者注意到自己的语言缺口
- 先输入再输出 = 先有素材再有能力

### 最佳实践

**第一个月（输入期）：**
- 背范文 → 不是逐字背，是内化结构和表达
- 拆结构 → 理解每段的功能
- 提取关键表达 → 积累高分词汇和句型
- 举一反三 → 用同样结构写不同话题

**第二个月（输出期）：**
- 限时写 → 20min Task 2 / 15min Task 1
- 批改 → 用 `/ielts-writing` 四维评分
- 背改写版 → 内化 AI 的改写
- 再写 → 巩固提升

## 6. 万能故事覆盖（Universal Story Coverage）

### 科学依据

- 记忆的自我参照效应：与自己相关的内容记得更牢
- 故事比孤立信息更容易记忆（叙事记忆优势）
- 一个故事可以灵活变通覆盖多个话题（迁移学习）

### 最佳实践

**少量个人故事覆盖高频 Part 2：**

| 故事 | 覆盖话题 |
|------|---------|
| 旅行经历 | 城市/地方/旅行/开心经历/和朋友做的事 |
| 影响你的人 | 朋友/家人/老师/佩服的人/帮助过你的人 |
| 学会的技能 | 礼物/拥有的东西/技能/爱好/有用的 app |
| 难忘的经历 | 成功/失败/挑战/改变想法的经历 |
| 一本书/电影 | 书/电影/电视节目/了解的话题/新闻 |

**关键原则：**
- 用中文想自己的真实经历 → AI 翻译成英文
- 记自己的故事比记别人的容易得多
- 故事要具体（名字、地点、时间、感受）
