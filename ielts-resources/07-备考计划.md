|----|------|---------|

| 4 | 冲刺 | C19-20 模拟考 + 查漏补缺 + 保持状态 |

## 6 周标准计划（适合基础 5.5-6，目标 7）

| 周 | 重点 | 每日安排 |
|----|------|---------|

## 8 周完整计划（适合基础 5-5.5，目标 7-7.5）

### 第 1 周：诊断 + 摸底

| 日 | 任务 | 时长 |
|----|------|------|
| 一 | 摸底：做一套完整真题（听读写 + 口语录音） | 3h |
| 二 | 诊断：用 `/ielts-diagnose` 分析成绩 | 30min |
| 三 | 词汇：背第 1 组 15 词 + 语境长难句 | 45min |
| 四 | 阅读：做第 1 套 + 四步法精读 | 1h45min |
| 五 | 听力：做第 1 套 + 精听错题 | 1h30min |
| 六 | 写作：背 1 篇范文 + 拆结构 | 1h |
| 日 | 口语：用 `/ielts-speaking` 生成万能故事 | 1h |

### 第 2-3 周：基础夯实期

| 日 | 主科目 | 具体任务 |
|----|--------|---------|
| 一 | 听力 | 做一套 + 精听错题 + 1.25x 跟读 |
| 二 | 阅读 | 做一套 + 四步法精读 |
| 三 | 听力 | 做一套 + 精听错题 + 1.25x 跟读 |
| 四 | 阅读 | 做一套 + 四步法精读 |
| 五 | 听力 | 做一套 + 精听错题 + 1.25x 跟读 |
| 六 | 写作 | 背范文 + 拆结构 + 举一反三（输入期） |
| 日 | 写作+口语 | 限时写一篇 + 批改 + 口语 Part 1 练习 |

每日固定：词汇 45min（20 新词 + 20 复习 + 5 入库）+ 口语 20min

### 第 4-5 周：强化提升期

变化点：
- 听力从 1.25x → 1.5x
- 写作从输入期 → 输出期
- 阅读开始练题型专项
- 口语开始练 Part 3 框架

| 日 | 主科目 | 具体任务 |
|----|--------|---------|
| 一 | 听力 | 做一套 + 精听 1.5x + 错题模式分析 |
| 二 | 阅读 | 做一套 + 四步法 + 题型专项（判断题） |
| 三 | 听力 | 做一套 + 精听 1.5x + 错题模式分析 |
| 四 | 阅读 | 做一套 + 四步法 + 题型专项（匹配题） |
| 五 | 听力 | 做一套 + 精听 1.5x + 错题模式分析 |
| 六 | 写作 | 限时写 Task 2（20min）+ 批改 + 背改写版 |
| 日 | 写作+口语 | 限时写 Task 1（15min）+ Part 3 框架练习 |

### 第 6-7 周：冲刺期

变化点：
- 开始做 Cambridge 19-20（最新真题）
- 写作每天一篇
- 听力能 1.5x 听懂
- 口语用 ChatGPT Voice 模拟考

### 第 8 周：考前最后一周

| 日 | 任务 | 时长 |
|----|------|------|
| 一 | C20 Test 1 模拟（严格计时） | 3h |
| 二 | 复盘 + 错题回顾 + 生词最后过一遍 | 2h |
| 三 | C20 Test 2 模拟（严格计时） | 3h |
| 四 | 复盘 + 口语万能故事最后过一遍 | 2h |
| 五 | 写作高频错误复习 + 范文背诵 | 2h |
| 六 | 轻松复习，不刷新题 | 1h |
| 日 | 考试 | — |

## 每日时间分配参考

| 任务 | 时长 | 说明 |
|------|------|------|
| 词汇 | 30-45min | 20min 新词 + 20min 复习 + 5min 入库 |
| 主科目 | 60-90min | 听力/阅读交替 |
| 口语 | 20-30min | 每天练一点 |
| 写作 | 60min | 输入期背范文 / 输出期限时写 |
| **总计** | **2.5-3.5h** | |

---

## 关键原则

1. **精读 > 刷量** — 一套题做透比十套题刷过有用
2. **语境背词 > 孤立背词** — 结合阅读背词记得牢
3. **1.5x 精听 > 大量泛听** — 把一篇听到滚瓜烂熟
4. **背改写版 > 自己瞎写** — 输入够了才能输出好
5. **数据驱动 > 感觉驱动** — 用 `/ielts-dashboard` 看数据

---

## 如何与 Skill 系统配合

1. 用 `/ielts-diagnose` 生成个人化计划（基于你的当前水平调整）
2. 每天按计划执行，用对应 skill 训练
3. 用 `/ielts-dashboard` 追踪进度
4. 根据数据调整计划（如果某科进步快，可以把时间让给其他科）
