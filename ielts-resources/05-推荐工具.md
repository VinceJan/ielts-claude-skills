# 雅思备考推荐工具

## AI 工具

### 1. Claude Code + IELTS Skill（核心工具）

**用途：** 写作批改、阅读分析、口语素材、词汇训练、进度追踪

**优势：**
- 本地运行，数据存在自己电脑上
- 8 个 Skill 协同工作
- 跨会话记忆，持续追踪进度
- 24 小时可用

**使用方式：**
```
/ielts          → 入口
/ielts-writing  → 写作批改
/ielts-reading  → 阅读精读
/ielts-listening → 听力分析
/ielts-speaking → 口语素材
/ielts-vocab    → 背单词
/ielts-diagnose → 诊断规划
/ielts-dashboard → 看进度
```

### 2. ChatGPT 语音模式

**用途：** 口语陪练、模拟考官

**优势：**
- 实时对话，纠正语法/词汇/发音
- 可以模拟 Part 1-3 完整考试流程
- 随时可用，不怕丢脸

**使用方式：**
- 把口语话题喂给 ChatGPT
- 让它扮演雅思考官
- 每次对话后复盘：哪里卡壳了？哪些表达可以更好？

### 3. Gemini Live

**用途：** 口语陪练（ChatGPT 的替代方案）

**优势：**
- 免费
- 支持多语言切换
- 可以在对话中切换中英文

### 4. NotebookLM

**用途：** 听力训练

**使用方式：**
1. 把口语素材/阅读文章喂给 NotebookLM
2. 生成两个人的对话或一个人的播客
3. 听这些生成的音频练听力
4. 可以调节语速

## 使用建议

### 工具组合（推荐）

```
核心：Claude Code + IELTS Skill（写作/阅读/听力/词汇/规划）
口语：ChatGPT Voice 或 Gemini Live（每天练 20 分钟）
听力：NotebookLM（生成播客练听力）
背词：Anki 或 Skill 内置的间隔重复
真题：ieltsolve.com（在线做 Cambridge 15-20）
```

### 不建议

- 不要用太多工具，2-3 个核心工具就够
- 不要只用 AI 评分，要交叉验证
- 不要忽略真题练习，AI 工具是辅助不是替代
