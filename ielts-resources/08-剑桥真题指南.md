# 剑桥雅思真题使用指南

---

## 真题概览

| 册数 | 难度 | 建议用途 |
|------|------|---------|
| Cambridge 1-10 | ★☆☆ 较简单 | 不建议使用（太旧，难度不匹配） |
| Cambridge 11-14 | ★★☆ 中等 | 基础练习期使用 |
| Cambridge 15-18 | ★★★ 标准 | **主力练习册**（备考时做） |
| Cambridge 19-20 | ★★★ 标准 | **考前模拟册**（最后两周做） |

**建议：从 Cambridge 15 开始练，15-18 做备考练习，19-20 留到考前模拟。**

---

## 每册结构

每册 Cambridge IELTS 包含：

| 部分 | 数量 | 说明 |
|------|------|------|
| Listening | 4 个 Test | 每个 Test 4 个 Section，40 题 |
| Reading (Academic) | 4 个 Test | 每个 Test 3 篇文章，40 题 |
| Writing | 4 个 Test | 每个 Test 2 个 Task |
| Speaking | 4 个 Test | 每个 Test 3 个 Part |

**6 册（15-20）× 4 个 Test = 24 套完整真题**

## 练习安排

### 备考期（Cambridge 15-18）

每周做 3 套阅读 + 3 套听力：

| 日 | 科目 | 真题 | 时间 |
|----|------|------|------|
| 一 | 听力 | C15T1 | 做题 30min + 精听 1h |
| 二 | 阅读 | C15T1 | 做题 60min + 精读 45min |
| 三 | 听力 | C15T2 | 做题 30min + 精听 1h |
| 四 | 阅读 | C15T2 | 做题 60min + 精读 45min |
| 五 | 听力 | C15T3 | 做题 30min + 精听 1h |
| 六 | 阅读 | C15T3 | 做题 60min + 精读 45min |
| 日 | 写作+口语 | 写作一篇 + 口语练习 | 2h |

### 考前期（Cambridge 19-20）

最后两周，严格模拟考试：

| 日 | 任务 | 时间 |
|----|------|------|
| 一 | C19T1 完整模拟（听读写） | 3h |
| 二 | 复盘 + 错题分析 | 2h |
| 三 | C19T2 完整模拟 | 3h |
| 四 | 复盘 + 错题分析 | 2h |
| 五 | C19T3 完整模拟 | 3h |
| 六 | 复盘 + 最后查漏补缺 | 2h |
| 日 | 休息或轻松复习 | 1h |

## 真题使用原则

1. **不要一开始就做 Cambridge 19-20** — 留到考前模拟
2. **每套题做完必须精读/精听** — 不要只刷题不沉淀
3. **计时做题** — 模拟考试节奏
4. **错题归档** — 用 Skill 系统自动记录到 `~/.ielts/`
5. **不要一次做太多** — 每天 1 套听力 + 1 套阅读就够了

---

## 如何与 Skill 系统配合

1. 做完真题后，把题目和答案喂给对应的 Skill
2. `/ielts-reading` 分析阅读错题 + 生词入库
3. `/ielts-listening` 分析听力错题 + 精听任务
4. `/ielts-writing` 批改写作
5. `/ielts-dashboard` 追踪正确率趋势
