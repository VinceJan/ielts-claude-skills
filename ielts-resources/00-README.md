# 雅思备考资料库

本目录包含雅思备考相关的高质量资料，与 IELTS Skill 系统配合使用。

## 资料清单

| 文件 | 内容 | 用途 |
|------|------|------|
| 01-考试结构.md | 听力/阅读/写作/口语的考试格式、题型、时间 | 了解考试 |
| 02-评分标准.md | 四科 Band 5-9 详细 descriptor + 官方文档链接 | 理解评分 |
| 03-口语题库.md | 2026 年 Part 1/2/3 高频话题整理（第三方/社区来源，需定期刷新） | 口语准备 |
| 04-写作话题.md | Task 2 高频话题 + Task 1 图表类型 | 写作准备 |
| 05-推荐工具.md | AI 工具 + 学习资源 + 免费练习网站 | 工具选择 |
| 06-科学学习法.md | 间隔重复、精读、语境背词等方法的科学依据 | 方法论 |
| 07-备考计划.md | 4 周/6 周/8 周计划模板 | 规划 |
| 08-剑桥真题指南.md | Cambridge 15-20 的使用建议和练习安排 | 真题使用 |

## 如何与 Skill 系统配合

- `/ielts-diagnose` 生成计划时会参考 07-备考计划.md
- `/ielts-speaking` 生成素材时会参考 03-口语题库.md
- `/ielts-writing` 出题时会参考 04-写作话题.md
- `/ielts-vocab` 背词时会参考 06-科学学习法.md 的间隔重复规则
- 所有 skill 的评分标准都基于 02-评分标准.md

## 来源

来源分层：
- 考试结构、评分与官方样题：优先使用雅思官方网站（ielts.org）、British Council、IDP、Cambridge University Press & Assessment
- 真题练习：Cambridge IELTS 系列
- 学习方法：二语习得和认知科学研究，按证据强弱使用
- 口语/写作题库：第三方/社区整理，适合作为练习素材，不应当作官方预测
